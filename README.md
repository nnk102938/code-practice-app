# コード記述練習アプリ

Pythonコードの入力・実行・判定を行う学習用Webアプリです。  
初心者が「理解できてもコードが書けない」という課題を解決することを目的として開発しました。

---

## 主な機能

- Pythonコード入力
- コード実行・正誤判定
- 問題ランダム出題
- 学習進捗・スコア管理

---

## 使用技術

- Python
- Flask
- HTML / CSS / JavaScript

---

## 工夫した点

- exec()実行時に使用可能な組み込み関数を制限
- sessionによる学習進捗管理
- print()出力のキャプチャによる結果判定
- 問題データと判定処理の分離

---
